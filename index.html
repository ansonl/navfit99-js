    FAQ

    1. What is NAVFIT99?
    • NAVFIT99 is an open source project that allows you to view and edit NAVFIT98A database (.accdb) files. This website uses a publicly accessible instance of NAVFIT99 on the internet which is intended for evaluation and testing purposes only.
    2. What is NAVFIT98A?
    • NAVFIT98A is a program that allows people to organize and edit U.S. Navy Evaluations and Fitness Reports. NAVFIT98A is available for Windows. See here for more information.
    3. Is there a warranty for NAVFIT99?
    • NO. NAVFIT99 is released under MIT License.
